## â–¶ï¸ How to Run the Project

You can run the project using any of the following methods:

### **1. Live Server (VS Code Extension)**

Best option for development.

1. Install **Live Server** extension in VS Code.
2. Right-click `index.html` â†’ **Open with Live Server**.
3. Your project will open at `http://127.0.0.1:5500`.

---

### **2. HTMLPreview Service** (Online Preview)

Useful when browsing the project online or sharing on GitHub.

1. Upload your project to GitHub.
2. Go to the repository.
3. Replace the URL with:

   * `https://htmlpreview.github.io/?<your-github-file-url>`

Example:

```
https://htmlpreview.github.io/?https://github.com/username/repo/blob/main/index.html
```

---

### **3. Python Local Server** (Quick & Simple)

If you have Python installed:

#### **Python 3**

```
python -m http.server 8000
```

Then open:

```
http://localhost:8000
```

#### **Python 2**

```
python -m SimpleHTTPServer 8000
```
